## Getting Started

### Develop

```shell
yarn dev
# or
npm run dev
```
If single entry file open [http://localhost:1234](http://localhost:1234), if mutiple entry files, need specifying file name open [http://localhost:1234/index.html](http://localhost:1234/index.html)

### Build

```shell
yarn build
# or
npm run build
```

### Clean

```shell
yarn clean
# or
npm run clean
```
